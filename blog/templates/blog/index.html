Home

Hello, and welcome to my website! My name is Michael Rose. I grew up Atlanta, studied at Georgia Tech, and currently work in software and quality engineering for Calendly.

I enjoy staying fit, playing chess, reading, traveling, and programming in my free time. The Internet has been a wonderful resource for me, and I want to start contributing to it. I created this website as a place to share information with others and to work on personal projects with that goal in mind. If you have any questions or comments about the website, please let me know through the GitHub project by clicking on the link in the footer of any page. If you'd like to contact me otherwise, please complete this web form.

Current Events

As of September 2019, I have just launched this site, so their isn't much going on besides this homepage. My first project was to create the site itself, which I did using Django because I am a Python guy. One of the more common use cases of Django is to create a blog application, so I have added one and it can be accessed via the top navigation panel. Check it out to see what I decided to post about! Here are some of the other projects I have in mind for the site right now, and you can always see the most up-to-date version on GitHub.

  • Create a registration portal and a web form to contact me via email
  • Develop a chess server in Python and integrate it into the site
  • Do something cool with fantasy football stats that I can share with my league

Credits

This site is powered by Django and Bootstrap. It is hosted on Digital Ocean, and the source code is on GitHub. I would like to extend a special thank you to the contributors for their efforts in maintaining and documenting these frameworks.
